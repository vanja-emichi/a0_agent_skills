## Exit Criteria (must ALL be true before proceeding to /test)
- [ ] All planned tasks implemented
- [ ] All unit tests pass
- [ ] No TODO comments remaining in new code
- [ ] Code compiles/builds without errors

## Anti-Rationalization Table
| Rationalization | Reality |
|---|---|
| "I'll test it all at the end" | Bugs compound. Test each increment. |
| "It works on my machine" | Prove it with automated tests, not manual checks. |
| "This refactor is too small to test separately" | Small untested changes become large untested codebases. |

## Red Flags
- More than 100 lines written without running tests
- Multiple unrelated changes in one increment
- Build broken between increments
