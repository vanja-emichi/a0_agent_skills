## Exit Criteria (must ALL be true before proceeding)
- [ ] OWASP Top 10 reviewed for scope

## Anti-Rationalization Table
| Rationalization | Reality |
|---|---|
| "Security review is overkill for this change" | One vulnerability can compromise the entire system. |
| "We'll fix security issues later" | Security debt compounds faster than technical debt. |
| "Automated scans are sufficient" | Automated scans miss business logic flaws and auth issues. |

## Red Flags
- Only automated scan, no manual review
- Critical findings deferred without justification
- No input validation checked
- No authentication/authorization tested
